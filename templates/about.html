About

pysoso is a lightweight RSS feed reader, written in flask.
it mirrors the functionality provided by rososo.

rather than presenting you with the most recent content from the sites that you follow, it simply lets you know when they have been updated. you can then read the content of these sites as it is intended: on the site itself.

shortcuts and helpers

pysoso supports a number of keyboard shortcuts to speed up common tasks. these include:

  • e: toggle editing on/off
  • a: add bookmark
  • c: hide/show controls
  • f: hide/show fresh bookmarks
  • s: hide/show stalebookmarks
  • 1, 2, 3 ..: select tags (hover mouse over tags to see shortcuts)
